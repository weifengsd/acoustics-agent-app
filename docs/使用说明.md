# Acoustics Agent Studio 使用说明

## 1. 简介
Acoustics Agent Studio (`acoustics-agent-app`) 是一个基于 `acoustics-agent` 的交互式水声学仿真可视化 Web 平台。它提供了一个具有现代“毛玻璃”暗黑风格的图形界面，让海洋学家和水声工程师能够以最直观的方式设计物理波导、配置仿真环境，并实时计算与查看声射线轨迹及相干传播损失场。

## 2. 界面概览
![Acoustics Studio 主界面 - Munk 信道声线图](./munk_ray_paths.png)

界面主要分为两大区域：
1. **左侧边栏 (Sidebar)**：用于加载经典声场预设环境、选择底层计算引擎模式（如纯 Python 射线追踪），以及查看系统加载状态。
2. **主操作区 (Main Workspace)**：用于微调环境参数、运行仿真、发送自然语言指令以及可视化各项计算结果。

---

## 3. 核心功能与操作指南

### 3.1 环境预设与声速剖面 (Environment & SSP)
- **快速加载预设**：在左侧边栏的 `Load Preset Scenario` 下拉菜单中，您可以一键选择预设的基准海洋波导场景（如 Munk 深海声道、Pekeris 浅海波导、表面混合层声道等）。选择后，工作区的参数会自动重置以匹配该物理环境。
- **动态声速剖面设计**：在主界面的数据表中，您可以双击单元格直接修改各个深度（Depth）对应的声速（c）。修改后，下方的折线图会实时动态更新。

### 3.2 物理波导与几何配置 (Waveguide & Geometry)
在 `🌊 Water & Bottom Environment` 区域，您可以精确调整：
- **海底深度 (Bottom Sea Depth)**：水体的最大深度。
- **声源参数**：设定声源的发射深度和声波频率（Hz）。
- **海底声学属性**：展开高级设置，可以自定义海底的压缩波速（Compressional Speed）、密度和衰减系数，从而构建复杂的声学弹性海底模型。
- **发射与接收阵列**：在几何配置区设定声线的起始俯仰角区间（Launch Angles）和射线密度。

### 3.3 运行仿真计算 (Simulation)
参数配置完毕后，点击操作中心绿色的 **✨ Run Ray Tracing Simulation** 按钮。系统将瞬间调用后端的 Python 求解器完成计算，并显示成功提示。

### 3.4 结果分析与可视化 (Analytics Workspace)
仿真完成后，可以通过下方并排的多个选项卡深度分析结果：
- **📈 Ray Paths (射线路径)**：
  - 提供交互式的 Plotly 图表（支持平移、缩放、悬停查看详情）以及高保真度的 Matplotlib 静态科学制图，完美展现声线在海水中的折射与反射轨迹。
- **🌈 Coherent TL (相干传播损失)**：
  - 点击 **🌈 Compute Coherent TL Field**，系统将启动高斯束求和（Gaussian Beam Summation）算法，渲染并绘制出包含精细干涉图样（Striations）的相干声场热力图。
- **🔔 Arrivals Analysis (多径冲激响应分析)**：
  - 在接收器设定区域输入指定的范围（Range）和深度（Depth），点击 **Analyze Arrivals at Receiver Location** 按钮。
  - 系统会捕捉所有穿过该坐标的声线，并绘制出直观的“多径到达”冲激响应图表。您可以清楚地在数据表中看到每一条抵达路径的传播时间延迟、幅度、发射角与到达角。

### 3.5 AI 声学智能助手 (AI Copilot)
页面底部的 **🤖 AI Acoustics Copilot** 提供了一种革命性的控制方式。您只需在文本框中输入自然语言指令，例如：
> *"Load Pekeris waveguide"*
> *"Set source depth to 150m and frequency to 300Hz"*
> *"Change bottom sound speed to 1750 m/s"*

点击发送后，Copilot 会自动解析您的意图，并在瞬间完成界面对应参数的修改！

---

## 4. 配置导出
在 **📝 YAML Schema** 选项卡中，系统会实时将您当前的完整环境设计转化为 `acoustics-agent` 标准配置格式。您可以直接复制该 YAML 文本，用于离线命令行批量计算或其他自动化集成。
